# Prime Number Implementations

This repository contains implementations of prime number checking/generation algorithms in various programming languages. The goal is to demonstrate basic programming constructs and language-specific features for a common computational task.

## Languages Included:

- Python
- Go
- Rust
- C
- C++
- Java

## Build and Run Instructions

Below are the instructions to build and run the prime number programs for each language.

### Python

This project includes a Python script for prime number operations.

**How to Run:**

```bash
python prime_number.py
```

### Go

This project includes a Go program for prime number operations.

**How to Build:**

```bash
go build -o out/primego prime.go
```

**How to Run:**

```bash
./out/primego
```

### Rust

This project includes a Rust program for prime number operations.

**How to Build:**

```bash
rustc main.rs -o out/main_rust
```

**How to Run:**

```bash
./out/main_rust
```

### C

This project includes a C program for prime number operations.

**How to Build:**

```bash
gcc primes.c -o out/primesc
```

**How to Run:**

```bash
./out/primesc
```

### C++

This project includes a C++ program for prime number operations.

**How to Build:**

```bash
g++ primes.cpp -o out/primescpp
```

**How to Run:**

```bash
./out/primescpp
```

### Java

This project includes a Java program for prime number operations.

**How to Compile:**

```bash
javac Primes.java -d out
```

**How to Run:**

```bash
java -cp out Primes
```
